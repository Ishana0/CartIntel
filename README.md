# 📊 CartIntel
**An End-to-End Unsupervised Machine Learning Project**

## 🎯 Project Objective
The goal of this project is to help a retail mall understand its customer base by grouping individuals with similar spending behaviors and income levels. By identifying these "Customer Personas," the marketing team can move away from generic advertising and toward **Data-Driven Personalization**.

---

## 🚀 The Data Science Workflow
1.  **Exploratory Data Analysis (EDA):** Analyzed the distribution of customer income and spending scores to identify initial patterns.
2.  **K-Means++ Clustering:** Applied the Scikit-Learn K-Means algorithm to segment the data points into 5 distinct groups.
3.  **Business Logic Mapping:** Developed a custom Python script to translate mathematical cluster centers into actionable business labels (e.g., "Target," "Sensible," "Careful").

---

## 📈 Visual Results & Key Findings
Based on the analysis, the customer base was segmented into the following five profiles:

| Segment | Profile Description | Marketing Strategy |
| :--- | :--- | :--- |
| **Target** | High Income, High Spend | **Priority:** Exclusive previews, loyalty rewards, and luxury ads. |
| **Careful** | High Income, Low Spend | **Focus:** High-value promotions to encourage spending. |
| **Standard** | Mid Income, Mid Spend | **Retention:** General brand awareness and seasonal discounts. |
| **Spendthrifts** | Low Income, High Spend | **Urgency:** Trend-based social media ads and flash sales. |
| **Sensible** | Low Income, Low Spend | **Value:** Budget-friendly bundles and essential-only offers. |
